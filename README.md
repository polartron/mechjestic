## Game Overview

SteamDeckBuilder is a roguelike deck-building card game where players:
- Build and customize their mech with cards
- Explore procedurally generated overworld levels
- Battle enemies in turn-based card combat
- Collect loot and perks to strengthen their build
- Progress through endless mode floors
- Customize their character, mech, starter deck, difficulty, and game mode

## Core Systems

### Battle System
Turn-based card combat where players use cards to attack enemies, equip items, and activate abilities. Features include:
- Turn management
- Attack costs and overpower system
- Card drawing and deck management
- Engine fuel system
- Ability activation
- Rift effects (Enduring Rift at level 10+)

[Read more →](Battle-System.md)

### Card System
Comprehensive card system with multiple card types:
- **Instant Cards**: One-time use cards
- **Equip Cards**: Equipment that modifies stats
- **Hero Cards**: Special hero equipment with unique abilities
- **Perk Cards**: Permanent upgrades
- **Customization Cards**: Character, Mech, Starter Deck, Difficulty, Game Mode

[Read more →](""Card-System.md)

### Overworld System
Procedurally generated overworld where players:
- Explore tile-based maps
- Encounter enemies, treasures, and special tiles
- Navigate using pathfinding
- Interact with various tile types

[Read more →](""Overworld-System.md)

### Enemy System
Dynamic enemy system with different roles and behaviors:
- **Aggressive**: Chases and pushes player
- **Strategist**: Tactical positioning and pushes
- **Resilient**: Moves every other turn
- **Repairing**: Support role, stays behind
- **Utilitarian**: Support role, stays behind strong enemies
- **Disruptive**: Standard chase behavior
